# ECOCSTEM
**Ecosystem x Computer Science**

## Site Link
- https://alenicakariz.github.io/eco-cs-tem/

## About the Project
This website explores how Computer Science can contribute to environmental conservation. 
This is built as an academic activity using **HTML, CSS, and JS**, and is organized into three sections: 
**Awareness, Detection, and Prevention** with each highlighting ways on how developers and technology can contribute to protecting ecosystems.

## Importance Notice
- This website is built for academic purposes only.
- It does not represent any real organization or program.
- The technologies, systems, and tools described in this site may already exist, be in development, or not yet exist in the real world.
- All descriptions are written for educational and illustrative purposes and do not claim technical accuracy.
